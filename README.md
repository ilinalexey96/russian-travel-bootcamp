Проект студента Яндекс.Практикума по направлению "Frontend-разработчик.Bootcamp" Алексея Ильина.
## Описание проекта: 
Проектная работа по адаптивной верстке. На разрешениях: 320, 768, 1024, 1280.

* Позиционирование
* Псевдоклассы
* Трансформации
* Flexbox
* Grid
